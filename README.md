Java Number Guessing Game

This is my first Java project developed during my DecodeLabs Internship.

About the Project
The Number Guessing Game is a simple console-based Java application. The program generates a random number, and the user tries to guess it. If the guess is too high or too low, the program provides hints until the correct number is guessed.

Features
- Generates a random number
- Takes user input
- Gives "Too High" and "Too Low" hints
- Displays a success message when the correct number is guessed

Technologies Used
- Java
- VS Code
